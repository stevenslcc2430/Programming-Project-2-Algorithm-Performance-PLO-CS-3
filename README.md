# Build/Run Instructions

# 1 - Prerequisites

* Python 3.10+
* An IDE (such as Visual Studio Code)

# 2 - Run the Program

* The main driver can be found in driver.py inside the scr/ folder
* Navigate to scr/ and run:
  ```
  python driver.py
  ```
* Results will print to the console and save as CSV files in scr/output/

# 3 - Comparison Definition

* A comparison is counted each time two array elements are compared against each other
* Loop bounds and index checks are not counted
